# CodePocket 📁

Библиотека готовых заготовок кода, созданная на Flask.

![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![Flask](https://img.shields.io/badge/flask-2.0+-green.svg)
![SQLAlchemy](https://img.shields.io/badge/SQLAlchemy-ORM-red.svg)

## 📖 Описание

**CodePocket** — это веб-платформа для разработчиков, позволяющая хранить и структурировать заготовки кода.

## ✨ Особенности

- 🗂 **Трехуровневая структура**: Четкое разделение (например, Python -> Телеграм боты -> Aiogram).
- 👤 **Система профилей**: Личный кабинет автора со статистикой публикаций и комментариев.
- 📋 **Copy-to-Clipboard**: Быстрое копирование кода прямо из карточки превью или со страницы поста.

## 🚀 Установка и запуск

### Предварительные требования

- Python 3.8 или выше
- Установленные зависимости (Flask, Flask-SQLAlchemy, Flask-Login, Flask-WTF)

### Запуск проекта

1. Установите необходимые библиотеки:
   pip install -r requirements.txt

2. Запустите Flask-приложение:
   python main.py

3. Или запустите .exe файл

## ⚙️ Техническая часть

### Архитектура базы данных (SQLAlchemy)
- **User** — модель пользователя с хешированием паролей и системой статусов (creator).
- **Language / Topic / Library** — связанные таблицы (One-to-Many) для реализации классификации.
- **CodeSnippet** — основная модель заготовки кода, привязанная к библиотеке и автору.
